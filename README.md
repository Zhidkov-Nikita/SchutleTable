## Таблица Шульте
Таблица Шульте — простая и полезная игра для тренировки внимания, скорости реакции и концентрации. Найдите числа по порядку от 1 до 25 как можно быстрее!

## Как играть
Нажмите кнопку "Новая игра" в правом верхнем углу экрана

Найдите и тапните числа по порядку: сначала 1, затем 2, 3... и так до 25

Время засеклось автоматически — попробуйте побить свой предыдущий результат!

## Особенности
Автоматическая смена фона под тему вашего устройства (тёмная/светлая) при запуске приложения

Приложение сохраняет только последний результат, а не лучший за всё время

## Совместимость
Минимальная версия: Android 5.0 (Lollipop)

Целевая версия: Android 13.0 (API 33)

Работает на всех современных Android устройствах!
